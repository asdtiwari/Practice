Inline Tag / Block level Tag

Those tags which give line break automatically (implicitly) i.e., without using <br> tag are known as block level tags
Example: <big>, <p>, <pre>, heading tags, etc.

Those tags which don't give line break automatically (implicitly) i.e., to give break, we've to use <br> tag are known as Inline tag.
Example: anchor <a> tag

Paired Tag / Unpaired Tag

Paired Tag: Such Tag which has seperate opening (<tag>) and closing (</tag>) known as paired tags.
Example: <p>, <html>, <center>, <a>, <pre>

Unpaired Tag: Such Tag which has only opening tag and no closing tag i.e., closing tag is not mandatory.
Example: <br>, <hr>, etc.

<img> Tag

It is used to display the images on the webpage.
It is of type Unpaired tag i.e., closing is not required.
"src" (Source) attribute we need to use with <img> tag to give the path of image.
<img> tag is inline tag.
The image should be present in the same folder on which you are working or on other way we can copy the path of image directly and paste it in our source code using "src" attribute
"height" and "width" are also attributes.

<audio> Tag

How to add audio on webpage?
1. Download one audio and save in the same folder on which you are working.
2. Use <audio> tag along with "src" attribute to give path of audio.

<audio> tag is of type paired tag.
It is of type inline tag.
<audio> tag requires "controls" attributes to give the control like (play, pause, speed, etc.)
