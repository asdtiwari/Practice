Cascading Style Sheet (CSS)

CSS is used to layout the webpage using CSS properties.

Basic CSS Properties:

  • font-size
  • font-style
  • color
  • background-color
  • text-align

Terminologies:

<tag attribute="property:value;">
  • Tag
  • Attribute
  • Property
  • Value

Types of CSS:

  1. Inline CSS
  2. Internal CSS
  3. External CSS
Inline CSS:
  • It is used to apply unique style to html element.
  • Example: 
    <tag attribute="property:value;">
Internal CSS:
  • It is used to apply styling to single html document/page.
  • Here, we use style as tag i.e., <style> </style>
  • An internal CSS is defined in the head section of an HTML page, within style tag.
  • Asterisk (*) is used to specify every elements present in html page
  • Example: 
    <!DOCUMENT html>
    <html>
        <head>
            <title> Internal CSS </tilte>
            <style>
                h2, h3, p, pre{color: darkmagenta; 
                font-family: cursive;
                fontsize: 20px;
                }
            </style>
            </head>
        <body> 
            ...
        </body>
    </html>
External CSS:
  • The external stylesheet is used when we want to make changes on multiple html documents.
  • In External css we are creating seperate stylesheet and we can link this stylesheet to multiple html documents by using the following element:
      <link href="stylesheetname.css" rel="stylesheet">
  • rel: This attribute defines the relationship between a linked source and the current document. "rel" specifies the relationship between two documents. rel is short form of relation.
  • link: link is an empty element which contains only attribute and link is used to link external stylesheet to html documents.
  • href: Hyper reference, this attribute is used to specify the location of the linked document.
  • Example of CSS file
    • h3,p{color:blue;
font-family: cursive;}

Note:

  • <h2 style="display: inline;"> is a css property which makes the block level html tags as inline tags.
  • if consecutive two elements are block level then we have to apply this property in both.

Selectors

are used to select html elements, you want to style.
  • Element Selector
  • Group Selector
  • Id Selector
  • Class Selector
  • Universal Selector

Element Selector

  • Whenever we are selecting html element by using element name to give the styling to it are called as element selector
  • Example:
    <html>
	<head>
		<style>
			p{color:blue}
		</style>
	</head>
	<body>
		<p> ... </p>
	</body>
</html>

Group Selector:

  • It is used to select html group to give same style definition.
  • Example: 
    <html>
	<head>
		<style>
			p, h3, h1, pre{color:blue}
		</style>
	</head>
	<body>
		<p> ... </p>
		<h3> ... </h3>
		<h1> ... </h1>
		<pre> ... </pre>
	</body>
</html>

ID Selector:

  • It is used to select html element by using Unique ID
  • This Unique ID is given by using id attribute.
  • Example: 
    <html>
	<head>
		<style>
			#one{color:blue}
			#two{color:blue}
			#three{color:blue}
			#four{color:blue}
		</style>
	</head>
	<body>
		<h3 id = "one"> ... </p>
		<h3 id = "two"> ... </h3>
		<h3 id = "three"> ... </h1>
		<h3 id = "four"> ... </pre>
	</body>
</html>
  • ID should not start with numeric value
  • While selecting an element using id we have to use "#"
  • ID is more strong, fast and specific than Class.

Class Selector:

  • It is used to select html element by using element class name.
  • This class is given by using class attribute.
  • Example: 
    <html>
	<head>
		<style>
			.one{color:blue}
			.two{color:blue}
			.three{color:blue}
			.four{color:blue}
		</style>
	</head>
	<body>
		<h3 class = "one"> ... </p>
		<h3 class = "two"> ... </h3>
		<h3 class = "three"> ... </h1>
		<h3 class = "four"> ... </pre>
	</body>
</html>
  • Class should not start with numeric value
  • While selecting an element using class we have to use "."

Universal Selector:

  • It selects all the html elements of the page
  • Example: 
    <html>
	<head>
		<style>
			*{font-family: cursive;
			background-color: aqua;
			}
		</style>
	</head>
</html>

Border:

  • It is a CSS property.
  • Example 1:
    <html>
	<head>
		 ... 
	</head>
	<body>
		<h3 style="border: 10px solid red;
			height:200px;
			width: 200px;> 
			...
		</h3>
	</body>
</html>
  • Example 2: 
    <html>
	<head>
		 ... 
	</head>
	<body>
		<h3 style="border-width: 10px;
			border-color: red;
			border-style: solid;
			height:200px;
			width: 200px;> 
			...
		</h3>
	</body>
</html>
  • CSS Border Style
    • dotted defines a dotted border
    • dashed defines a dashed border
    • solid defines a solid border
    • double defines a double border
    • groove defines a 3D ridged border. The effect depends on the border-color value
    • ridge defines a 3D ridged border. The effect depends on the border-color value.
    • inset defines a 3D inset border. The effect depends on the border-color value.
    • outset defines a 3D outset border. The effect depends on the border-color value.
    • none defines no border.
    • hidden defines hidden border.
  • We can use multiple border-style too. E.g.: "border-style=dotted dashed solid double;"
  • border is divided into four parts: top, bottom, left & right. We can explicitly define each parts of border as per our requirements. Example: "border-top-style: dashed;"
  • If we want to apply width, style and color then we can directly tell by saying "border: width_value style_value color_value" instead of telling one by one.
  • border-radius
    • border-radius is a property used to rounded the border
    • We have also four more parts of border viz top-left, top-right, bottom-left & bottom-right to set the radius only.
    • If we apply single value to property "border-radius: value1;" then all the parts automatically set to be value1 only
    • If we apply double value to property "border-radius: value1 value2;" then top-left and bottom-left will assigned to value1 and rest to value2.
    • If we apply triple value to property "border-radius: value1 value2 value3;" then value1 would be assigned to top-left & bottom-right while value2 and value3 would be assigned to top-right and bottom-left respectively.
    • If we apply four value to property "border-radius: value1 value2 value3 value4;" then value1, value2, value3 and value4 would be assigned to top-left, top-right, bottom-left and bottom-right respectively.
    • If we don't provide the value for border-color then by default it adopt the color of text

  • Span is an inline tag which is used to display the content in one line.
  • We can use it with <p> </p> and <pre> </pre>
  • The span tag is used for the grouping of the inline html elements.
  • We can use span tag as a container to group inline elements together so we can style or manipulate them with javascript.
