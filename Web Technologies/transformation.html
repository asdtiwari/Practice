Transform Margin & Padding

Tranform properties: scale, skew, translate, rotate
Using transform properties we can transform, rotate, skew and scale an element.
  • scalex(n) property is used to scale html element n time in x axis.
  • scaley(n) property is used to scale html element n time in y axis.
  • scale(m, n) property is used to scale html element m times in x axis and n times in y axis.
  • skewX(m) and skewY(m) are used to skewed the html elements in m degree in x and y axis respectively.
  • Providing positive deg in rotate() rotates html elements in clockwise direction and negative deg in rotate() rotates html elements in anti-clockwise direction.
  • margin-left property is used to provide margin from the left side.
  • margin properties give from the html elements not from the browser's border.
  • Similarly, there is something known as padding which moves the content inside the html elements
  • We can directly pass the alignment in all direction using margin: top_px left_px
  • Similarly for padding also, padding: top_px left_px
  • Negative values for padding properties are not allowed.
  • Negative values for margin propety are used to go opposite to the functionality of the respective property
