  • Instead of defining explicitly the animation's name, duration and iteration count, we can just use "animation: pattern1 5s infinite"
  • the values of style tag can not be commented by pair of <!--- and -->, we have to use /* and */
animation-name
It specifies the animation name which we have created using the followed with animation name, and we can give any animation
animation-duration
This property specifies the duration of the animation on an element.
animation-iteration-count
This property sepcifies the total iteration of the repetation of the animation to be infinte.
animation
By Using this property we can specify "animation-name", "animation-duration", "animation-iteration-count" at once.
